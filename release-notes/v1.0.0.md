# ember.term v1.0.0

First stable release of the standalone terminal version of ember.

## Highlights

- Native C renderer built for lightweight terminal animation instead of Neovim window APIs
- Three scenes: `fire`, `lava`, and `spiral`
- `widget`, `fullscreen`, and `print-frame` modes
- Built-in benchmark mode for measuring frame cost directly
- Release archives for macOS and Linux, with a signed macOS archive prepared during release packaging and source build fallback via `make`

## Notes

- Lava still looks a little odd, but it is interesting enough to keep as a scene.
- If you prefer the editor-integrated version, see `ember.nvim`.
- If a prebuilt archive does not fit your environment, build locally with `make`.
